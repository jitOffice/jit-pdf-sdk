## 简介

`jit-pdf` 是一套面向业务系统的 PDF 预览与编辑 SDK，支持：

- Vue 组件接入
- 浏览器挂载式接入
- UMD `<script>` 直引
- 本地 `IndexedDB` 模式
- HTTP 文件服务模式
- 中文界面、蓝色主题、批注与脱敏

## 安装

```bash
npm install jit-pdf vue
```
